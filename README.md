\# Fears to Fathom — Fan Website Redesign



An unofficial fan-made website redesign concept for the Fears to Fathom horror anthology.



Built with React, TypeScript, Vite, ReactBits, Font Awesome, HTML and CSS.



\## Features



\- Horror-inspired landing page

\- Animated Dock navigation

\- Spotlight episode cards

\- Latest release highlight

\- Orbiting tech stack section

\- Official links/community section

\- Responsive design

\- Fan-made disclaimer
